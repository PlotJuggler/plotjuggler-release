Main elements of the GUI

  1. The application can have 1 or more Windows. They are arrange in a single MainWindow and 0 or more SubWindows.
  2. A window can have 1 or more Tabs.
  3. A tab can have 1 or more Plots. Each plot will show a Legend, if not disabled.
  4. A plot can have 0 or more Curves.
  5. The MainWindow has a searchable List of Curves on the left side.
